# 🏥 MedClear

MedClear is a healthcare cost transparency platform that helps users compare hospitals, estimate treatment costs, discover emergency facilities, and make informed healthcare decisions.

## ✨ Features

- 🔍 Search hospitals by treatment or procedure
- 📍 Filter by city and PIN code
- 🏥 Compare hospitals side-by-side
- 🚑 Emergency hospital finder
- 📊 Treatment cost transparency
- ⭐ Hospital ratings and clarity scores
- 📱 Responsive user interface
- 🖼️ Real hospital images
- 📄 Detailed hospital information

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Node.js
- JSON Database

## 🚀 Run Locally

Clone the repository

```bash
git clone https://github.com/AARAYHARSHIT/MEDCLEAR.git
```

Go to the project folder

```bash
cd MEDCLEAR
```

Run the server

```bash
node server.mjs
```

Open

```
http://127.0.0.1:8000
```
